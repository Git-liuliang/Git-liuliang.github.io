cut learning

cut以某种方式按照文件的行进行分割

cut参数：

  • -b 按照字节选取
  • -c 按照字符选取
  • -d 自定义分隔符，默认为制表符
  • -f 与-d一起使用，指定显示的区域

example：

cat /etc/passwd | cut -b1-3 #取每行的第1-3字字节
cat /etc/passwd | cut -b1-3,4-5,5-8 #获取每行的第1-3 4-5 5-8字节
cat /etc/passwd | cut -d ":" -f 3 #以":"分割取第三段
cat /etc/passwd | cut -d " " -f 3 #以" "分割取第三段

awk learning

awk [options] [program] [file]

awk -F ":" '{print $1}' file   #按照":"将每行分割后，取每行的第一个字段，$0表示整行内容，$1表示1行，以此类推
awk  '{$1="123";print $0}' test #按照空格将每行分割，将每行第一个字段替换成123，并打印出整行内容
awk 'NR==2' file        #输出awk执行文件后所读取的数据行数为2的那一行

2.查看http并发请求数与tcp连接状态

netstat -n | awk '/^tcp/ {++S[$NF]} END {for (a in S) print a,S[a]}'
netstat -n | awk '/^tcp/{print $NF}' | sort | uniq -c
uniq -c 在每列旁显示该行重复出现的次数

3.获取nginx的log访问数量前10的ip

cat access.log |grep awk {print $1} | sort -rn | uniq -c | head -10

tip:
1.cut的功能是分割每行的内容，这个awk几乎可以代替
2.awk的功能就是分割每行内容，也可以使用NR或者FNR来取到具体某一行的内容，功能比较全
3.sed也可以取到某一行的数据，同时也可以使用正则匹配具体内容
