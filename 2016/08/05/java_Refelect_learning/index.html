java反射总结

反射原理

  • 在框架开发中，都是基于配置文件开发。在配置文件中配置了类，可以通过反射得到类中的所有内容
  • 反射原理分析：
    1.首先需要把java文件保存到本地磁盘.java
    2.编译java文件，生成.class文件
    3.使用jvm把class文件通过类加载器加载到内存中
    4.class文件在内存中使用class类表示
    5.当使用反射时，首先需要获取到class类，得到这个类后，就可以得到class文件里的所有内容
    6.其中包括属性、构造方法、普通方法，分别对应类Filed、Constructor、Method

反射操作

  • 获取class类
    1.类名直接.class 

          `Class class1 = Person.class;  `

    2.对象.getClass() 

          `Class class2 = new Person().getClass(); `

    3.Class.forName 

         `` Class class3 = Class.forName("com.reflect.www.Person");  ``

  • 不使用new获取类的实例且操作实例属性 

       Class class3 = Class.forName("com.reflect.www.Person");  
   Person p3 = (Person) class3.newInstance();  
   p3.setName("zhangsan");  
   System.out.println(p3.getName());

  • 使用反射操作有参构造方法

          Class class3 = Class.forName("com.reflect.www.Person"); 
      Constructor cs = class3.getConstructor(String.class,String.class,int.class);
      Person person = (Person) cs.newInstance("zhangsan","gray",12);
      System.out.println(person.getName()+person.getBirth()+person.getAge());

  • 使用反射操作属性

      Class pw = Class.forName("com.reflect.www.Person");
  Person person = (Person) pw.newInstance();
  Field field = pw.getDeclaredField("name");
  field.setAccessible(true);
  field.set(person,"GUn");
  System.out.println(field.get(person));

  • 使用反射操作普通方法

      Class pw = Class.forName("com.reflect.www.Person");
  Person person = (Person) pw.newInstance();
  Method md = pw.getDeclaredMethod("setName", String.class);
  md.invoke(person,"fangyunchang");
  System.out.println(person.getName());

    **需要注意操作静态方法时，md.invoke(person,”fangyunchang”);中的第一个参数需要设置为null
