python+influxdb+grafana监控系统

Tips：
python收集数据，起udp client发送消息
python 起udp server接收数据，定时刷新通过graphite协议将数据发送到influxDB
grafana来展示influxDB的数据

安装influxdb

curl -sL https://repos.influxdata.com/influxdb.key | sudo apt-key add -
echo "deb https://repos.influxdata.com/ubuntu trusty stable" | sudo tee /etc/apt/sources.list.d/influxdb.list
apt update
apt install influxdb
sudo service influxdb start

配置influxdb

详细配置参数信息参考官网https://docs.influxdata.com/influxdb/v1.7/administration/config/
这里主要开启graphite的配置，用于接收监控客户端传来的数据，其他配置默认

[[graphite]]
  # Determines whether the graphite endpoint is enabled.
  enabled = true
  database = "grafana"
  retention-policy = ""
  bind-address = ":2003"
  protocol = "tcp"
  consistency-level = "one"
  templates = [
    ".host.measurement*",
    # Default template
    # "server.*",
  ]
  # templates参数可以将.host.measurement*之前的配置变成tag插入表中，在grafana中可以用作tag筛选

安装grafana

详细参照grafana官网http://docs.grafana.org/installation/debian/

curl https://packages.grafana.com/gpg.key | sudo apt-key add -
Update your Apt repositories and install Grafana

sudo apt-get update
sudo apt-get install grafana
On some older versions of Ubuntu and Debian you may need to install the apt-transport-https package which is needed to fetch packages over HTTPS.

sudo apt-get install -y apt-transport-https

sudo service grafana-server start

配置grafana

将data sources配置成上面的influxdb的地址

python向influxdb发送数据

开启graphite协议，接收的数据格式如下：

“metric_path value timestamp\n”

metric_path: arbitrary namespace containing substrings delimited by dots. The most general name is at the left and the most specific is at the right.
value: numeric value to store.
timestamp: epoch time.
messages must end with a trailing newline.
multiple messages maybe be batched and sent in a single socket operation. each message is delimited by a newline, with a trailing newline at the end of the message batch.

Example message:

"foo.bar.baz 42 74857843\n"

Let’s look at some (Python 2.7) code for sending data to graphite…

#!/usr/bin/env python

import platform
import socket
import time


_SERVER = '0.0.0.0'
_PORT = 2003
DELAY = 15  # secs


def get_loadavgs():
    with open('/proc/loadavg') as f:
        return f.read().strip().split()[:3]


def send_msg(message):
    print 'sending message:\n%s' % message
    sock = socket.socket()
    sock.connect((_SERVER, _PORT))
    sock.sendall(message)
    sock.close()


if __name__ == '__main__':
    node = platform.node().replace('.', '-')
    while True:
        timestamp = int(time.time())
        loadavgs = get_loadavgs()
        lines = [
            'system.%s.loadavg_1min %s %d' % (node, loadavgs[0], timestamp),
            'system.%s.loadavg_5min %s %d' % (node, loadavgs[1], timestamp),
            'system.%s.loadavg_15min %s %d' % (node, loadavgs[2], timestamp)
        ]
        message = '\n'.join(lines) + '\n'
        send_msg(message)
        time.sleep(DELAY)
